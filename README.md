# ESP32 Soccer Robot with Ball Collector 🤖⚽

A high-performance Soccer Robot firmware featuring a specialized **Ball Collector** mechanism. This project was engineered for competitive environments, focusing on speed, precision control, and robust power management.

## 🏆 Achievements
* **2nd Place Winner** at the **4th Engineering Forum & 1st International Forum** - Mansoura National University (MNU).
* Representing: **IEEE MNU Student Branch**.

## 🚀 Key Features
* **Ball Collector Mechanism:** Dedicated logic and mechanical design for high-speed ball capture and handling.
* **PS4 Controller Integration:** Real-time, responsive wireless control via Bluetooth (DualShock 4).
* **Stable Power Management:** Integrated **Step-Down (Buck) Converters** to provide consistent voltage for the ESP32, preventing system resets during high-current motor surges.
* **Optimized Maneuverability:** Precise PWM control for smooth drivetrain movement and scoring.

## 🛠 Technical Specifications
* **Microcontroller:** ESP32.
* **Power Module:** Step-Down Converter (e.g., LM2596) for logic power stability.
* **Motor Drivers:** High-current DC drivers (L298N / BTS7960).
* **Communication:** Bluetooth Low Energy (BLE).
* **Chassis:** Custom mechatronics design with a specialized collector system.
